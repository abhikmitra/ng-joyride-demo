ng-joyride

This plugin is inspired by the Jquery Joyride plugin.This also adds lot of new functionality. And has no dependency on Jquery Joyride.


Dependencies Version
Jquery 1.8+
Bootstrap (optional) 3.0
Angular 1.2+

Jquery

The Jquery Dependency is required for the selector mechanism

Bootstrap

This is not a hard requirement.If you supply your own templates then this is not required.But if you use the default templates, then its required.

Angular

I have tested this for 1.2 + and the latest build of 1.3
